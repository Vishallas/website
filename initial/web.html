1. Inputs & Outputs in Arduino

Arduino basically has two types of pins called digital & analog pins in which digital pins has 1's and 0's data ie HIGH or LOW

Code: 

        void setup(){
            pinMode(2,OUTPUT); // making the pin 2 as output
            pinMode(3,INPUT); // making the pin 3 as input
        }

2. Using LED with Arduino

Now coming to using a digital pin as output a led is connected to arduino's digital pins through a 330 ohm resistor and other leg of LED to GND of arduino

Code: 

        void setup(){
            pinMode(13,OUTPUT); // making the pin 13 as output
        }
        void loop(){
            digitalWrite(13,HIGH); // giving the pin 13 voltage
        }
